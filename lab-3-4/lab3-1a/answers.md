What is Spring?

- It is an opensource light weight framework for building enterprize level java applications.

What is Spring Boot?

- one of the widely adopted spring frameworks for building spring application without providing configuration. It follows convention over configuration approach.

What is the relation between Spring platform and Spring Boot?

- Spring boot is built on top of spring.

What is the relation between Spring platform and Spring framework?

- Spring platform mainly focus on the spring core, and spring framework encompose the entire ecosystem.

What is Dependency Injection and how is it done in the Spring platform/framework?

- It is a design pattern where we pass dependencies of objects rather than creating them internally with new keyword. which increase code reusualibity and decrease coupling

What is Inversion of Control (IoC) and how is it related to Spring?

- It is a principle where the creatin of an object is moded outside of the application to external framework.
