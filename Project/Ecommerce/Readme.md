# ECOMMERCE

Spring boot application for online e-commerce application. Current functionalities include

- Authentication
- Product management
- Cart and order management

# Instruction for local development

- Make sure you have docker installed and running
- `git clone https://github.com/NatnaelSisay/ASD.git`
- Move to project directory `cd Project/Ecommerce`
- run application : `docker-compose up`
